🔗 Integration Bridge Core

Universal Integration Layer for External Platforms

Overview

Integration Bridge Core provides a generic, reusable framework for connecting external platforms with Odoo. It replaces multiple platform-specific integrations with a single, unified endpoint that routes to platform-specific handlers.

Supported Platforms

📱 Chatwoot 📝 Typebot 💬 Evolution API 🔄 n8n 🤖 AI Agents ➕ Extensible

Key Features

🔐 Universal Token System

Manage authentication for all platforms in one place. Generate secure tokens, set expiration, enable IP whitelisting.

📊 Complete Logging

Audit trail for every inbound and outbound request. Full request/response capture with duration tracking.

🚀 Unified Endpoint

Single API endpoint that routes to platform-specific handlers. Simplifies integration architecture.

📤 Outbound Queue

Reliable message delivery with automatic retry, priority support, and exponential backoff.

🎯 Platform Routing

Automatic routing to Chatwoot, Typebot, Evolution, or n8n handlers based on payload.

🔧 Easy Extension

Base controller class makes adding new platforms simple. Just implement handler method.

API Endpoints

Unified Inbound
POST /bridge/inbound
Single entry point for all platforms
Health Check
GET /bridge/inbound/health
Monitor system status

Example Request

curl -X POST https://your-odoo.com/bridge/inbound \ -H "X-Bridge-Token: your_token_here" \ -H "Content-Type: application/json" \ -d '{ "platform": "chatwoot", "event_type": "message_created", "data": { "name": "Error title", "description": "Error description", "reporter": { "phone": "+201234567890", "name": "Customer Name" }, "chatwoot": { "conversation_id": "12345" } } }'

Quick Setup

  1. Install Module: Apps → Search "Integration Bridge Core" → Install
  2. Create Token: Integration Bridge → Bridge Tokens → New → Generate Token
  3. Configure Platform: Update your n8n workflow with new endpoint and token
  4. Test: Send test request via curl or n8n
  5. Monitor: Integration Bridge → Monitoring → Integration Logs

Architecture

  • Base Controller: Reusable authentication and logging
  • Unified Router: Platform detection and routing
  • Token System: Centralized authentication with IP whitelisting
  • Request Logger: Complete audit trail
  • Outbound Queue: Reliable delivery with retry
