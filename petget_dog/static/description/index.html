Petget: Dog

Dog-specific data on top of the Petget animal profile.

What this module does

  • Breed catalog — 200+ AKC breeds pre-loaded, grouped (Sporting, Hound, Working, Terrier, Toy, Non-Sporting, Herding).
  • Dog fields — breed, coat type, and AKC registration number, shown only when the animal's species is Dog.
  • Puppy-stage fields — temporary ID and collar colour.
  • Breed column in the animal list for quick filtering.

Requires the free Petget: Core module.
